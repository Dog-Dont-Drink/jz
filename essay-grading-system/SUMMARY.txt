================================================================================
快速启动
================================================================================

1. 配置 Supabase:
   - 创建项目: https://supabase.com
   - 复制 URL 和 anon key
   - 编辑 .env 文件

2. 初始化数据库:
   - 执行 supabase/migrations/001_initial_schema.sql
   - 执行 supabase/migrations/002_storage_policies.sql
   - 创建 essay-images bucket (Private)

3. 部署 Edge Functions:
   supabase functions deploy ocr-recognize
   supabase functions deploy grade-essay

4. 配置 API Keys:
   supabase secrets set BAIDU_OCR_API_KEY=your_key
   supabase secrets set BAIDU_OCR_SECRET_KEY=your_key
   supabase secrets set DEEPSEEK_API_KEY=your_key

5. 启动开发服务器:
   npm run dev
